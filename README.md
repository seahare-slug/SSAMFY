# SSAMFY (**S**eungkyu **S**ongsub **A**nd **M**ovie **F**or **Y**ou)
How to Start
- Vue(front)
  - final-pjt/final-pjt-front
    - `$ npm install`
    - `$ npm run serve`
#
- Django(back)
  - final-pjt/final-pjt-back
    - `$ pip install -r requirements`
    - `$ python manage.py makemigrations`
